X Controller

Xbox controller guide for X/Twitter

Quick Start

  1. Open x.com or twitter.com.
  2. Click the page once to enable gamepad input.
  3. Use the controls below.

Controls

Left Stick Smooth scroll
D-pad Up/Down Previous / Next post (hold to repeat)
D-pad Left/Right Previous / Next post
A Like (double‑press to unlike)
X Reply
Y Repost / Undo repost
B Back / Close dialog
LB / RB Previous / Next post
Menu (Start) Open selected post
View (Back) Jump to top

Tips

  • If a stick doesn’t respond, click it (LS/RS) while moving to calibrate.
  • Selection is highlighted with a blue outline.
